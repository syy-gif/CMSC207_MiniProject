## 📋 Project Overview

A single-page personal web profile built for **CMSC 207**. The design follows a clean, minimalist editorial aesthetic using semantic HTML, a separated external CSS file, Bootstrap 5 for responsive layout, and vanilla JavaScript for interactivity.

The profile introduces **Syljie Canete**, a Masters of Information Systems student and security analyst, and includes sections for About, Hobbies & Interests, a Fun Fact interactive feature, and Contact.

## ✅ Features

- **Sticky Navigation Bar** — Smooth anchor-link scrolling with underline hover effect
- **Split Hero Section** — Full-height two-column layout with hero background image
- **About Me Section** — Profile photo, sidebar metadata, personal bio, and travel photo strip
- **Travel Strip** — Side-by-side photo strip (Macau, Dubai, Singapore) with hover zoom and label overlay
- **Hobbies & Interests Grid** — Six cards with real photos, grayscale-to-color hover effect
- **Fun Fact Button** — JavaScript interaction cycling through 8 personal facts with a reveal counter
- **Favorite Song Embed** — YouTube video embedded in the contact section
- **Contact Section** — Email, location, university, GitHub, and social media links
- **Scroll-Reveal Animations** — Elements animate into view using the Intersection Observer API
- **Responsive Layout** — Mobile-friendly via Bootstrap 5 grid and CSS media queries
- **Easter Egg** — Hidden interaction triggered by clicking the navbar brand five times
